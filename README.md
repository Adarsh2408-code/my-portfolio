## 🧠 Featured Projects

### 🏦 Bank Transaction Fraud Detection System
An end-to-end ML pipeline to detect fraudulent transactions using supervised learning.
- EDA to uncover patterns and handle class imbalance
- Feature engineering with Logistic Regression & Random Forest
- Evaluated with precision, recall, F1-score, and ROC-AUC
- Visualized fraud patterns using Matplotlib & Seaborn

**Tech:** Python · Scikit-learn · Pandas · Matplotlib · Seaborn

---

### 🌐 Social Network Analyzer
A Python-based tool for analyzing social graphs and generating recommendations.
- Data cleaning pipeline (duplicates, inactive users, conflicts)
- "People You May Know" using mutual-friend graph traversal
- "Pages You Might Like" via collaborative filtering logic
- Modular design, scalable to large datasets

**Tech:** Python · JSON · Graph Algorithms · Data Structures
